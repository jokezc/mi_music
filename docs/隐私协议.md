# 风花雪乐 隐私协议

**生效日期：2026年1月26日**  
**最后更新：2026年1月26日**

## 引言

欢迎使用风花雪乐（以下简称"本应用"或"我们"）。我们非常重视您的隐私保护和个人信息安全。本隐私协议旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

**重要提示：** 在使用本应用之前，请您仔细阅读并充分理解本隐私协议。当您点击"同意并继续"或开始使用本应用时，即表示您已充分理解并同意本隐私协议的全部内容。如果您不同意本隐私协议的任何内容，请立即停止使用本应用。

## 一、信息收集范围

### 1.1 设备信息

为了区分不同设备并提供基础服务，我们可能会收集以下设备信息：

- **设备唯一标识符**：Android 设备的 Android ID 或 iOS 设备的 identifierForVendor
- **设备型号**：设备制造商、型号、操作系统版本
- **应用版本信息**：当前应用版本号

**收集目的：**
- 用于区分不同设备，提供个性化服务
- 用于统计应用使用情况，改进产品功能
- 用于错误定位和问题修复

**隐私保护说明：**
- 设备唯一标识符不包含任何个人身份信息（如姓名、手机号、身份证号等）
- 我们不会将设备标识符与您的个人身份信息进行关联
- 设备标识符仅用于匿名统计，不会用于追踪您的个人行为

### 1.2 使用统计信息

为了了解应用使用情况并改进产品，我们可能会收集以下使用统计信息：

- **页面访问统计**：您访问的应用页面及访问时长
- **功能使用统计**：您使用的功能模块及使用频率
- **操作事件统计**：您在应用中的操作行为（如播放、暂停、切换设备等）

**收集目的：**
- 了解用户使用习惯，优化产品功能
- 分析应用性能，提升用户体验
- 识别热门功能，指导产品发展方向

**隐私保护说明：**
- 所有统计数据均为匿名化处理
- 不会记录您的登录信息（用户名、密码等）
- 不会记录您的音乐内容和个人偏好
- 统计数据仅用于产品改进，不会用于其他商业目的

### 1.3 错误信息

为了定位和修复应用问题，提升应用稳定性，我们可能会收集以下错误信息：

- **崩溃信息**：应用崩溃时的错误堆栈信息
- **异常信息**：应用运行时的异常错误信息
- **性能信息**：应用性能相关的技术指标

**收集目的：**
- 快速定位和修复应用问题
- 提升应用稳定性和用户体验
- 优化应用性能

**隐私保护说明：**
- 错误信息中不包含您的个人身份信息
- 不包含您的登录凭证（密码、Token等）
- 不包含您的音乐内容和个人数据
- 错误信息仅用于技术问题修复，不会用于其他目的

### 1.4 友盟统计服务

本应用集成了友盟（UMeng）统计服务，用于收集和分析应用使用数据。友盟可能会收集以下信息：

- **设备信息**：设备型号、操作系统版本、设备唯一标识符
- **应用信息**：应用版本、安装渠道
- **使用统计**：页面访问、事件统计、会话时长
- **错误信息**：崩溃日志、异常堆栈

**友盟隐私政策：**
- 友盟作为独立的第三方服务提供商，其隐私政策请参考：[友盟隐私政策](https://www.umeng.com/page/policy)
- 我们已与友盟签署数据处理协议，确保数据安全
- 友盟收集的数据仅用于统计分析，不会用于其他商业目的

**友盟数据使用说明：**
- 友盟收集的数据仅用于应用统计分析
- 数据存储在友盟服务器，受友盟隐私政策约束
- 我们不会将友盟数据与您的个人身份信息进行关联
- 您可以通过友盟提供的隐私设置控制数据收集（如适用）

### 1.5 权限申请

为了提供完整的功能体验，本应用可能需要申请以下系统权限：

- **网络权限（INTERNET）**：用于连接服务器，获取音乐资源
- **存储权限（READ_EXTERNAL_STORAGE / READ_MEDIA_AUDIO）**：用于读取本地音乐文件，支持音乐下载功能
- **前台服务权限（FOREGROUND_SERVICE）**：用于后台播放音乐
- **唤醒锁权限（WAKE_LOCK）**：用于保持设备唤醒，确保音乐正常播放

**权限使用说明：**
- 所有权限申请均基于功能需要，不会用于其他目的
- 您可以在系统设置中随时撤销已授予的权限
- 撤销权限可能会影响部分功能的正常使用

## 二、信息使用方式

### 2.1 使用目的

我们收集的信息仅用于以下目的：

1. **提供服务**：为您提供音乐播放、设备控制等核心功能
2. **改进产品**：分析使用数据，优化应用功能和用户体验
3. **问题修复**：定位和修复应用错误，提升应用稳定性
4. **安全保障**：检测和防范安全风险，保护应用和用户安全

### 2.2 使用原则

我们严格遵循以下信息使用原则：

- **最小化原则**：仅收集实现功能所必需的信息
- **匿名化原则**：尽可能使用匿名化或去标识化的数据
- **目的限制原则**：仅用于明确声明的目的，不用于其他用途
- **透明度原则**：向您清晰说明信息收集和使用情况

## 三、信息存储与保护

### 3.1 数据存储

- **本地存储**：部分数据（如登录配置、应用设置）存储在您的设备本地
- **服务器存储**：统计数据存储在友盟服务器，受友盟隐私政策约束
- **存储期限**：统计数据通常保留一定期限后自动删除，具体期限遵循友盟政策

### 3.2 数据安全

我们采取以下措施保护您的信息安全：

- **加密传输**：使用 HTTPS 等加密协议传输数据
- **访问控制**：严格限制数据访问权限，仅授权人员可访问
- **安全审计**：定期进行安全审计，及时发现和修复安全漏洞
- **数据备份**：定期备份重要数据，防止数据丢失

## 四、信息共享与披露

### 4.1 第三方服务

本应用集成了以下第三方服务：

- **友盟统计服务**：用于应用统计和错误上报
  - 服务提供商：友盟（北京友盟网络科技有限公司）
  - 隐私政策：[友盟隐私政策](https://www.umeng.com/page/policy)
  - 数据用途：应用统计分析、错误监控

### 4.2 共享原则

我们不会向第三方出售、交易或出租您的个人信息。仅在以下情况下，我们可能会共享信息：

1. **获得您的明确同意**
2. **法律法规要求**：根据法律法规、司法程序或政府要求
3. **服务提供商**：与为我们提供服务的第三方（如友盟）共享必要信息
4. **紧急情况**：为保护用户或公众的生命、财产等重大合法权益
